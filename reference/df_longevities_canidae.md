# Longevities for 133 Canidae species

Data frame containing Time of Speciation (TS) and Time of Extinction
(TE) for 133 Canidae species

## Usage

``` r
df_longevities_canidae
```

## Format

A data frame object with 133 rows and 3 columns:

- species:

  Species names

- TS:

  Time of Speciation

- TE:

  Time of Extinction
