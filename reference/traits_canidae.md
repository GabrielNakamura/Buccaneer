# Trait data for 133 Canidae species

Data frame containing five traits measured for 133 Canidae species

## Usage

``` r
traits_canidae
```

## Format

A data frame object with 133 rows and 5 columns, each row correspond to
a species name their respective traits:

- species:

  Species name

- diet:

  Numeric indicating the diet category of species

- LD1:

  Numeric indicating the level of hypercarnivore of species

- log_mass:

  Numeric indicating the mass of species in log scale

- diet_cat:

  Character indicating the diet category of species. hyper indicate
  hypercarnivores, meso indicates mesocarnivores and hypo hypocarnivores
