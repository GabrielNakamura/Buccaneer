# Occurrence records

Data frame containing spatial occurrence records of 133 species of
canidae in North America

## Usage

``` r
df_occurrence_canidae
```

## Format

A data frame object with 133 rows and 14 columns:

- species:

  Species name

- MaxT:

  Maximum (oldest) age estimate for each occurrence record

- MinT:

  the minimum (youngest) age estimate for each occurrence record

- lng:

  The longitude coordinate of the occurrence record

- lat:

  The latitude coordinate of the occurrence record

- Site:

  Name of the paleontological site location identifier

- max_low_res:

  Same as MaxT but with lower temporal precision

- min_lo_res:

  Same as MinT but with lower temporal precision

- site.char:

  Name of the paleontological site identifier

- midpoint:

  Age midpoint of occurrence record
